Setup Guide

    
The app stores data in your browser. To sync to Google Sheets and share with your manager, follow these steps.

    
1
Create a Google Sheet
Go to sheets.google.com and create a new sheet. Add this header row in row 1:
Date | Prospect | Series | Session | TotalSessions | Confidence | Objection | Delivery | Tips | Strengths | Fix | Timestamp

    
2
Open Apps Script and paste this code
In your Sheet click Extensions then Apps Script. Delete all existing code and paste exactly:
function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  if (!e || !e.parameter) {
    return ContentService
      .createTextOutput(JSON.stringify({status:'ok',message:'Script is live'}))
      .setMimeType(ContentService.MimeType.JSON);
  }
  var p = e.parameter;
  if (p.action === 'append') {
    sheet.appendRow([
      p.date||'', p.prospect||'', p.series||'',
      Number(p.session)||1, Number(p.total)||1,
      Number(p.confidence)||0, Number(p.objection)||0,
      Number(p.delivery)||0,
      p.tips||'', p.strengths||'', p.fix||'',
      new Date().toISOString()
    ]);
    return ContentService
      .createTextOutput(JSON.stringify({status:'ok'}))
      .setMimeType(ContentService.MimeType.JSON);
  }
  var data = sheet.getDataRange().getValues();
  return ContentService
    .createTextOutput(JSON.stringify({rows:data}))
    .setMimeType(ContentService.MimeType.JSON);
}

    
3
Deploy as Web App
Click Deploy, then New deployment, then Web app. Set Execute as: Me, Who has access: Anyone. Copy the URL and paste it in your browser. You should see {"status":"ok","message":"Script is live"} to confirm it works.

    
4
Connect the app
Click Add Entry and paste the Web App URL into the Google Sheets URL field. Hit Test to confirm a row appears in your Sheet.

    
5
Host on GitHub Pages
Upload index.html to a GitHub repository. Go to Settings, then Pages, then Branch: main, then / (root), then Save. Your tracker will be live at yourusername.github.io/repo-name within a minute.
