Setup Guide

This app works standalone using your browser storage. To also sync entries to Google Sheets (so you can share with your manager), follow these steps.

1

Create a new Google Sheet

Go to sheets.google.com and create a new sheet. Name it something like SE Performance Tracker. Add this header row in row 1:

Date | Prospect | Confidence | Objection Handling | Delivery | Tips | Strengths | Fix | Timestamp
2

Open Apps Script

In your Sheet, click Extensions → Apps Script. Delete any existing code and paste the script below exactly.

function doGet(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var p = e.parameter; if (p.action === 'append') { sheet.appendRow([ p.date, p.prospect, Number(p.confidence), Number(p.objection), Number(p.delivery), p.tips, p.strengths, p.fix, new Date().toISOString() ]); return ContentService .createTextOutput(JSON.stringify({status: 'ok'})) .setMimeType(ContentService.MimeType.JSON); } // Default: return all rows var data = sheet.getDataRange().getValues(); return ContentService .createTextOutput(JSON.stringify({rows: data})) .setMimeType(ContentService.MimeType.JSON); }
3

Deploy as Web App

Click Deploy → New deployment → Web app. Set Execute as to Me and Who has access to Anyone. Click Deploy and copy the Web App URL.

4

Paste URL into the app

Go to the Add Entry tab and paste your Web App URL into the Google Sheets field. From then on, every saved entry will also write a row to your Sheet automatically.

5

Host on GitHub Pages

Create a new GitHub repository. Upload this index.html file. Go to Settings → Pages → Branch: main → / (root) and save. Your tracker will be live at yourusername.github.io/repo-name within a minute.

6

Share with your manager

Send them the GitHub Pages URL for the live dashboard. Or share the Google Sheet directly with view-only access. Both work independently.
