Setup Guide

This app stores data in your browser. To also sync to Google Sheets so your manager can view entries, follow these steps.

1

Create a Google Sheet

Go to sheets.google.com, create a new sheet named SE Performance Tracker. Row 1 should have these headers:

Date | Prospect | Session | Confidence | Objection | Delivery | Tips | Strengths | Fix | Timestamp
2

Open Apps Script

In your Sheet: Extensions → Apps Script. Delete all existing code and paste this exactly:

function doGet(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); if (!e || !e.parameter) { return ContentService .createTextOutput(JSON.stringify({status:'ok',message:'Script is live'})) .setMimeType(ContentService.MimeType.JSON); } var p = e.parameter; if (p.action === 'append') { sheet.appendRow([ p.date||'', p.prospect||'', p.session||'', Number(p.confidence)||0, Number(p.objection)||0, Number(p.delivery)||0, p.tips||'', p.strengths||'', p.fix||'', new Date().toISOString() ]); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); } var data = sheet.getDataRange().getValues(); return ContentService .createTextOutput(JSON.stringify({rows:data})) .setMimeType(ContentService.MimeType.JSON); }
3

Deploy as Web App

Click Deploy → New deployment → Web app. Execute as: Me. Who has access: Anyone. Copy the URL and paste it into the Google Sheets URL bar in Add Entry.

4

Test it

Paste the URL into Add Entry, hit Test Connection, then check your Sheet for a row with prospect = Connection Test.

5

Host on GitHub Pages

Upload index.html to a GitHub repo. Go to Settings → Pages → Branch: main → / (root). Your tracker will be live at yourusername.github.io/repo-name.
